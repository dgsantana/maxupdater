maxupdater
==========

3dsMax plugin updater

Collection of python scripts with support for running as a service (pywin).

Current features:
- backup and restore mode (use a workstation for "backup" to central repo)
- file list
- env variables, get and set
- registry variables
- ini updating
- multiple versions
